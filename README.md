IntelliView 🎯
Overview

IntelliView is an AI-powered interview evaluation system designed to assess candidates efficiently and objectively. It analyzes communication skills, subject knowledge, and non-verbal cues to provide instant feedback and ratings.

Features
Speech and communication analysis
NLP-based answer evaluation
Facial expression recognition
Real-time feedback and scoring
Automated and unbiased evaluation system
Modules
User Module
User registration
Receive application number
Attend interview
Admin Module
Manage candidates
Schedule interviews
Evaluate performance
View results and reports
Tech Stack

Frontend: HTML, CSS, JavaScript
Backend: Python (Django)
Database: SQLite
Libraries: OpenCV, NumPy, PIL

Installation
Clone the repository
git clone https://github.com/Athulkrish27/IntelliView.git
Navigate to project folder
cd IntelliView
Create virtual environment
python -m venv venv
Activate environment (Windows)
venv\Scripts\activate
Install dependencies
pip install -r requirements.txt
Run the server
python manage.py runserver
How It Works
User registers and gets application ID
Admin schedules interview
Candidate attends interview
System analyzes speech, answers, and facial expressions
Final score is generated instantly
Objective

To build an intelligent system that reduces human bias, saves time, and improves recruitment efficiency.
